responsive-table-setup([$namespace])
âžœ Creates namespaced classes for responsive tables. Couple with mq() to create new responsive table breakpoints classes to go with them!

$namespace string (Null)
Media namespace ("small", "medium", "large", "xlarge") or any custom namespace.
NOTE: Passing nothing to this mixin will duplicate classes that already exist in Fuselage.
NOTE: Be sure to use UPPER BOUNDS, this is different than using width-setup().
Example SCSS
{% highlight scss %} // Blackberry responsive tables. Super duper important. @include mq(384px) {@include responsive-table-setup($namespace: "blackberry");} // Creates class to be used with responsive tables for this media size (blackberry-responsive). {% endhighlight %}
